Icon System

Icons are named with Apple SF Symbol strings and map to Material Icons by default — the exact vocabulary from the app's <IconSymbol>. iOS renders native SF Symbols; Android & web fall back to Material Icons, with a few FontAwesome brand glyphs for socials.

Custom service icons

Bespoke two-tone glyphs for the core services — drawn in brand pink over near-black #111827. Lifted from the app's CustomIcons.tsx. Use these (not Material) for the headline service tiles.
